scripts.main_menu.first_option()[source]

Function called in case if user entered first option in menu().

Note

  1. Gives a command to robot to move in user-defined coordinates (X, Y).

  2. Takes coordinate for X and checks whether it number or not using is_number_tryexcept() function. If it’s positive-negative then takes it as correct coordinate. Otherwise prints that entered coordinate is invalid.

  3. Takes coordinate for Y and checks whether it number or not using is_number_tryexcept() function. If it’s positive-negative then takes it as correct coordinate. Otherwise prints that entered coordinate is invalid.

  4. Checks if the /navigation service is available and blocks as long as it can’t reach the service.

  5. Creates the object which communicates with the /navigation server.

  6. If first_option returned True or False, then clearConsole() called for both cases. But results are checked in first_option.

scripts.main_menu.main()[source]

Main function which manages entered string-inputs from menu() function. Equates typed_input with returned string-input from menu() function.

  • Checks whether typed_input is one of the following cases. Otherwise, prints that option is not in the list.

  • In case while loop is not true anymore, then exit from the program.

scripts.main_menu.menu()[source]

Main menu function. After entering appropriate option, this string-input will be sent to main() to match with corresponding function. Clears console each time program is executed calling clearConsole() function.

Returns

Takes string-input and returns it. An option that user chose in the main menu.

Return type

input(string)

Note

After pressing one of the following key buttons corresponding mode will be executed:

1 — Autonomously reach X and Y coordinate.

2 — Drive the robot with the keyboard.

3 — Drive the robot with avoid collisions feature.

4 — Close the program.

scripts.main_menu.second_option()[source]

Function called in case if user entered second option in menu(). Provides users with a feature to self-control robot. Sends a 0 to second_option where it will run second option in case of 0, while in case of 1 it will run third option.

scripts.main_menu.third_option()[source]

Function called in case if user entered third option in menu(). Provides users with a feature to self-control robot. Sends a 1 to second_option where it will run third option in case of 1, while in case of 0 it will run second option.

Function to check distance from 3 different sides, which returns minimal range at the end.

Parameters

Returns

Returns array with minimum distance until the obstacle.

Return type

Note

  1. Bound angles on appropriate sides and store it in mid_d in appropriate order.

  2. Stores information about minimal range from 3 sides (right, front, left) in mid_d.

scripts.third_option.obstacle_avoidance(info)[source]

Function to check whether robot is too close to the obstacle.

Note

  1. Creates publisher to publish Twist to /cmd_vel.

  2. Takes array with minimal ranges from distance_control() function.

  3. If distance from right is less than border_th = 0.6, then it’s not possible to rotate to right further.

  4. If distance in front of the robot is less than border_th = 0.6, then it’s not possible to move robot forward further.

  5. If distance from left is less than border_th = 0.6, then it’s not possible to rotate to left further.

  6. Publish linear and angular values for X, Y and Z.
